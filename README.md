# 🎭 Playwright Automation Framework (Python)

This project is a test automation framework built using **Playwright**, **Python**, and **Pytest**.
It demonstrates end-to-end testing practices, scalable structure, and reporting.

---

## 🚀 Tech Stack

* Python
* Playwright
* Pytest

---

## ⚙️ Setup & Installation

Clone the repository:

```bash
git clone https://github.com/AlfredoMc95/Reto-playwright_automation-python.git
cd Reto-playwright_automation-python
```

Install dependencies:

```bash
pip install -r requirements.txt
playwright install
```

---

## ▶️ Run Tests

Run all tests:

```bash
pytest
```
---

## 📊 Playwright Report

After execution, open the Playwright report:

```bash
trabajar
```
